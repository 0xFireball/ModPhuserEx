ModPhuserEx

A free, open-source protector for .NET, .NET Core, and Mono

Download Preview

About

ModPhuserEx is a free, open-source protector for .NET applications. It is a fork of the now discontinued ConfuserEx project.

Features

ModPhuserEx supports all modern versions of the .NET Framework, Mono, and the new cross-platform .NET Core framework. It supports most of the protections youâ€™ll find in commerical protectors, and some more!

Protection List

  • Symbol renaming
  • WPF/BAML renaming
  • Control flow obfuscation
  • Method reference hiding
  • Anti debuggers/profilers
  • Anti memory dumping
  • Anti tampering (method encryption)
  • Embedding dependency
  • Constant encryption
  • Resource encryption
  • Compressing output
  • Extensible plugin API
